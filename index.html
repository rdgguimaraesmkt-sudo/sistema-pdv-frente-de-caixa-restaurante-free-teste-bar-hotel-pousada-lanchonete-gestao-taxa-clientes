Perguntas Frequentes (FAQ)

O sistema precisa de internet para funcionar?

Não. O AjudaAí possui um servidor HTTP embutido que funciona em rede local (offline). Você só precisa que o computador central (Caixa) e os celulares dos garçons estejam conectados ao mesmo roteador Wi-Fi do seu estabelecimento.

Como os garçons fazem os pedidos?

O sistema gera uma página web local (waiter.html). Os garçons acessam um endereço IP direto pelo navegador dos seus próprios celulares, sem precisar baixar nenhum aplicativo nas lojas.

Consigo criar minha conta sozinho após baixar?

Não. Prezamos pela segurança e estabilidade da sua operação. A ativação inicial, configuração do banco de dados e criação de administrador exigem meu suporte via AnyDesk para garantir que nada dê errado no seu dia a dia.
